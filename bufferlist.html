    Keybindings

    Press the cursor keys (Left / Right) to navigate the pages. Using the number keys (1, 2, 3, and so on), you can directly jump to the tab of the given number. Pressing them while holding CTRL pressed, will remove the given tab (e.g. CTRL + 3).

    Pressing CTRL + Shift + B is used to open the BuffTabs tab if it is not opened yet or jump to it if it is and you are in another tab. If you use it while in the BuffTabs tab, the tab will close. Finally, you can press F1 to show this help.
